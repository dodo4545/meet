# Meet App

A serverless, progressive web application (PWA) built with React using test-driven development (TDD) techniques. The app uses the Google Calendar API to fetch upcoming events.

## Features

- Filter events by city
- Show/hide event details
- Specify number of events
- Offline usage
- Add app shortcut to home screen
- Display charts visualizing event details

## Technologies

- React
- Google Calendar API
- AWS Lambda
- OAuth2 authentication
- Recharts for data visualization
- Jest and React Testing Library for testing
- Puppeteer for end-to-end testing

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

### Testing

```bash
npm test
```

### Build

```bash
npm run build
```
